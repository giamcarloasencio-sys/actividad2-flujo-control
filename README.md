# Actividad 2 - Flujo de Control 

## Descripción

Este programa fue desarrollado en C++ y permite calcular el Impuesto Sobre la Renta (ISR) de un empleado según su sueldo mensual.

El programa:

- Lee el sueldo mensual.
- Calcula el sueldo anual.
- Determina si aplica ISR utilizando la escala salarial de la DGII.
- Muestra el ISR mensual y el sueldo neto.
- Si el empleado está exento del impuesto, muestra 0 (No aplica).
